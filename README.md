# Agro Forte — agrinho2026

Este projeto é um site institucional para o Agro Forte, uma plataforma de práticas agrícolas inovadoras e sustentáveis. Ele apresenta um design moderno, responsivo e com foco na experiência do usuário tanto em desktop quanto em dispositivos móveis.

## O que o projeto faz

- Apresenta uma página inicial com uma seção *hero* clara e chamativa, destacando a proposta de valor do Agro Forte.
- Mostra informações sobre a missão da marca, os benefícios das práticas agrícolas sustentáveis e a economia circular.
- Exibe um conjunto de cards de práticas agrícolas inovadoras, cobrindo temas como plantio de precisão, agricultura regenerativa, sistemas agroflorestais, irrigação inteligente, logística local e economia circular.
- Inclui uma seção de gráficos interativos com botões de tabulação para alternar entre produção, consumo e sustentabilidade.
- Mostra métricas resumidas de impacto em cards compactos e visuais.
- Apresenta depoimentos de parceiros e produtores para transmitir confiança e resultados reais.
- Inclui uma seção de perguntas frequentes (FAQ) para esclarecer dúvidas comuns sobre as práticas agrícolas.
- Possui uma chamada para ação (call to action) convidando o visitante a conectar seu negócio a um futuro sustentável.
- Conta com um formulário de contato funcional e uma área de informações de contato, facilitando que o visitante entre em contato.

## Arquivos principais

- `index.html` — marcação HTML da página, com seções, navegação e conteúdo do site.
- `style.css` — estilos CSS para cores, tipografia, layouts responsivos e micro-interações.
- `script.js` — comportamentos interativos, como alternância de abas, animações de carregamento, contadores animados e menu mobile.

## Tecnologias usadas

- HTML5 para estrutura e semântica
- CSS para layout, responsividade, grid/flexbox e efeitos visuais
- JavaScript para interações dinâmicas e animações
- Chart.js para renderizar gráficos interativos

## Estrutura de navegação

O site inclui um menu fixo com links para as seções:

- Início
- Sobre
- Práticas
- Impacto
- Gráficos
- Depoimentos
- FAQ
- Contato

## Responsividade

O layout foi desenvolvido com uma abordagem *mobile-first* e usa `grid` e `flexbox` para se ajustar a diferentes larguras de tela, garantindo uma boa experiência em smartphones, tablets e desktops.

## Como usar

1. Abra `index.html` em um navegador.
2. Edite o conteúdo de `index.html` para personalizar textos, títulos e seções.
3. Ajuste as cores e tipografia em `style.css` conforme necessário.
4. Adicione ou modifique gráficos em `script.js` para refletir dados reais.

## Objetivo

Este projeto serve como uma página institucional de apresentação de uma empresa do agronegócio, permitindo demonstrar soluções sustentáveis, métricas de impacto e facilitar o contato com potenciais clientes e parceiros.
